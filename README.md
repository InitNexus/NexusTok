### Introduction
Welcome to the **NexusByte Analyst** repository. An all-in-one data, analytics and statistics collection / lookup tool for all major social media platforms. This repository also contains a database of collected information in the form of a website. The goal is to display what information can be collected from profiles on each platform and how it can be used.  

While the website displays collected information, some human interaction is needed for the tool. While the tool doesn't require API keys in order to work, it is suggested as more information can be collected (plus it's more reliable).

> [!NOTE]
> Information on how to obtain API keys isn't listed in this description as it's all documented within the download.

### Usage (Tool)
1. Download the tool [here](https://google.com/404) or from the [releases](https://google.com/404) section
2. Download and install Python 3.11.12 (ensure that it's added to PATH if running on Windows)
3. Install the requirements via the dependancy installer
4. After everything is installed, restart your system and run the tool
5. Choose your platform of choice and enter a username

After you enter a username for your chosen platform, results will load (depending on your internet speed) and will be displayed. Please be aware that most of the tool is local. Some of the features may sound like they rely on outside interaction but they don't. Everything is operational from your system!
